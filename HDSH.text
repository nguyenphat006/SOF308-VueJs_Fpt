Bước 1: Cài đặt package cho vuejs dùng lệnh "npm install" (nhớ để đúng thư mục)
Bước 2: Cài Vue-router dùng lệnh "npm install vue-router@next"
Bước 3: Cài Vuex dùng lệnh "npm install vuex@next"
Bước 4: Cài yarn dùng lệnh "npm install --global yarn"
Bước 5: Dùng lệnh "npm run dev" hoặc "npm run serve" để chạy source
